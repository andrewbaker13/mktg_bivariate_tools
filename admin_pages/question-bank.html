How to Use Variables

  • Uniform: Random numbers in a range (e.g., B0 between 5-15)
  • Discrete: Random pick from list (e.g., brands: Nike, Adidas, Puma)
  • Array: Tied data matrix - each student gets one column, all rows stay synchronized
    • Define: 3Ã—3 matrix with [product names, prices, warranties]
    • Access: {{ARRAY[0]}} for row 0, {{ARRAY[1]}} for row 1, etc.
    • Student A gets column 0, Student B gets column 1 - values stay tied!
  • Formula (Calculate): Compute answer from other variables
    • Basic: {{B0}}+{{B1}}*{{X}}
    • Math functions: Math.max(), Math.min(), Math.abs(), Math.round(), Math.floor(), Math.ceil(), Math.pow(), Math.sqrt()
    • Examples: Math.max({{q1}}, {{q2}}, {{q3}}), Math.abs({{value}} - {{target}}), Math.sqrt({{variance}})
  • Formula (Display): Show formula with values (e.g., "Sales = 8.2 + 1.5 Ã— X")
