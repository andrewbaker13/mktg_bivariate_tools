StudyFlow A/B Testing Comparison Tool

Compare two marketing variants with instant visual diagnostics, confidence intervals, and clear test guidance.

Context: As a Data Analyst for StudyFlow (a student study-habit SaaS), your task is to compare the observed conversion rates for two marketing channels: the current partner Influencer Alpha (Control, P1) and the new partner Influencer Beta (Variant, P2).

This interactive tool helps you analyze A/B test results using the observed sample sizes and conversion rates. It computes confidence intervals, visualizes uncertainty with fan bands, and performs a two-proportion z-test to assess whether the difference is statistically meaningful.

Core Test (two-proportion z-test with hypothesized difference Δ₀): $$z = \frac{P_2 - P_1 - \Delta_0}{\sqrt{\dfrac{P_1(1-P_1)}{n_1} + \dfrac{P_2(1-P_2)}{n_2}}} \quad\text{(Wald / unpooled)}$$
Null hypothesis: \(H_0:\ \Delta = \Delta_0\). By default \(\Delta_0 = 0\) (no difference).

Confidence Interval for the difference (Δ = P2 − P1): $$\Delta \pm z_{1-\alpha/2}\sqrt{\frac{P_1(1-P_1)}{n_1}+\frac{P_2(1-P_2)}{n_2}}$$
